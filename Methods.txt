auth.checkPhone
Возвращает статус регистрации системы для переданного на нее номера телефона
После отправки номера телефона


auth.sendCode
Отправляет текстовое сообщение с кодом подтверждения, необходимым для регистрации, на номер телефона
После отправки номера телефона при выборе подтвердить по СМС

auth.sendSms
Принудительно отправить SMS-сообщение на указанный номер телефона
После наажатия "выслать код" при выборе подтвердить по СМС

auth.sendCall
Выполняет голосовой вызов переданного телефонного номера, в котором синтезированный голос повторяет код подтверждения из текстового сообщения
После отправки номера телефона при выборе подтвердить по звонку

auth.signUp
Регистрирует подтвержденный номер телефона в системе
После звонка или ввода кода

auth.signIn
Авторизовать пользователя в системе на основе подтвержденного номера телефона
При получении статуса, что номер зарегистрирован

auth.logOut
Деактивирует пользователя в системе
После нажатия "выйти"

auth.sendInvites
Сохраняет информацию о том, что текущий пользователь отправил приглашения текстовых сообщений своим незарегистрированным контактам
После отправки приглашений

auth.resetAuthorizations
Завершение разрешенных сеансов.


auth.exportAuthorization
Возвращает данные, необходимые для копирования разрешения в другой центр обработки данных


auth.importAuthorization
Авторизовать пользователя в системе на основе ключа, переданного из центра обработки данных


auth.bindTempAuthKey 
Привязывает временный ключ авторизации к постоянному ключу авторизации
